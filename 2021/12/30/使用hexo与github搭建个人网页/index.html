使用hexo与github搭建个人网页

Posted on 2021-12-30

本文介绍如何搭建个人博客平台，使用的工具是Hexo框架和Github。

1 Hexo

hexo是一个快速简介和高效的静态博客框架，使用它，可以快速美观的把markdown格式的文件转换为漂亮的网页格式。

1.1 Hexo的安装

hexo是基于Node.js 运行的，因此我们首先需要下载并安装Node.js，下载地址为：Node.js，按照提示安装即可。

在安装完Node后，打开命令行（window用户使用win+R，输入cmd），在命令行中输入node -v，如果出现了版本号，就说明安装成功了，此时就可以安装Hexo了，在命令行中输入：

3 Hexo使用过程中遇到的问题

3.1 hexo d出现错误

hexo d出现错误，代表的是本地文件上传到github上出现了问题，解决方案有两步：

  • 首先将_config.yml中deploy中https://github.com/yourgitname/yourgitname.github.io.git替换为：git@github.com:yourgitname/yourgitname.github.io.git，如果已经是这这个就不同换了；

  • 删除你的博客文件夹内的 .deploy_git 文件夹后再试
